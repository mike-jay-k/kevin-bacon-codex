# Kevin Bacon Codex

This repository contains a simple single-page application that checks whether two actors have appeared in the same movie using [The Movie Database (TMDb) API](https://developer.themoviedb.org/reference/intro/getting-started).

Open `index.html` in a browser. You'll need a TMDb API key, which you can obtain for free by creating an account on TMDb and requesting an API key. Replace the `YOUR_API_KEY_HERE` placeholder in the file with your key.
